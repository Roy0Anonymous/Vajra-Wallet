# Vajra-Wallet

This is a sample wallet demonstrating the use of Bitcoin Dev Kit and Lightning Dev 
Kit for a native iOS Swift app.

Works on Regtest and TestNet Networks.

## Features
* [x] On-chain BDK Wallet
* [x] Connect to a Peer
* [x] List Peers
* [x] Get Node ID
* [x] Open Channel
* [x] List Channels
* [x] Display Channel Balances
* [x] Send Payment
* [x] Receive Payment 
* [x] Close Channel
